# meme連コミュニティ スロット

Discord Coin Slot Lottery System

## 概要
Discordのチャット発言数に基づいてスロットを回せる抽選システムです。

## 使い方
1. `index.html` をブラウザで開く
2. Discordユーザー名と発言数を入力してプレイヤー追加
3. リストからプレイヤーを選択してスロットを回す
4. 3枚のコインが揃ったら当選！Xでシェアできます
